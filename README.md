# Reto Sophos Automatización

**Primer punto**: se desarrolla una automatización que ingrese a la URL https://www.vivanda.com.pe/, busqué un producto que se enceuntra en el feature, lo agrega al carrito de compras y valida que efectivamente se alla agregado correctamente.

**Segundo punto**: se desarrolla una automatización que consuma un Api Rest desde la página: https://gorest.co.in/ y consulte el id de un usuario, este id se encuentra en el feature. Por último valida que el nombre que retorna el recurso sea igual a que se espera en el feature.
